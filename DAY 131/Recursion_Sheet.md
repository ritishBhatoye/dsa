## Recursion Sheet (FAANG‑oriented)

**Legend**  
- **[⭐ Must-do]**: Core Striver / Love Babbar style questions, very common in MAANG/FAANG.  
- **[🔥 Very important]**: High-value patterns that repeat across many problems.  
- **[✅ Good practice]**: Warmups that build basic recursion thinking.  

1. Print 1 to N and N to 1 using recursion  *(5–10 min)*  — [✅ Good practice]  
2. Sum of first N natural numbers (recursive)  *(5–10 min)*  — [✅ Good practice]  
3. Factorial of N (recursive)  *(5–10 min)*  — [✅ Good practice]  
4. Power function \(a^b\) (linear recursion)  *(10–15 min)*  — [✅ Good practice]

6. Reverse an array / string using recursion  *(10–15 min)*  — [✅ Good practice]

9. Print all indices of a target in array (recursive)  *(15–20 min)*  — [✅ Good practice]  
10. Generate Fibonacci numbers using recursion  *(10–15 min)*  — [✅ Good practice]
