## Core Recursion Sheet (Minimal but Sufficient)

**Goal**: If you master only these questions, you will have covered almost every major recursion/backtracking pattern used in FAANG‑level interviews.

1. Print 1 to N and N to 1 using recursion  *(warmup – learn base/recursive case)*  
2. Reverse an array / string using recursion  *(pointer / index recursion)*  
3. Power function \(a^b\) with fast exponentiation  *(logarithmic recursion)*  
4. Basic Fibonacci with memoization  *(intro to recursion + DP)*  

5. Print all subsequences of an array / string  *(subset recursion template)*  
6. Subset Sum / check if subset with sum = K exists  *(classic include/exclude)*  
7. Generate all subsets (power set) including handling duplicates (unique subsets)  
8. Generate all permutations of an array / string (no duplicates)  
9. Unique permutations when input has duplicates  

10. Combination Sum I (unbounded choice)  
11. Combination Sum II (each element once, handle duplicates)  
12. Generate all valid parentheses of length 2N  

13. Rat in a Maze (all paths, simple grid)  
14. N‑Queens (print all solutions)  
15. Sudoku Solver (9x9 board)  
16. Word Search in a grid (existence of a given word)  

17. Tower of Hanoi (print all moves)  
18. Josephus problem (recursive solution)  

19. Height / depth of a binary tree (recursive DFS)  
20. Preorder / Inorder / Postorder traversals (recursive)  
21. Diameter of a binary tree (optimized recursion)  
22. Lowest Common Ancestor in a binary tree / BST  
23. All root‑to‑leaf paths with sum = K (path sum)  

24. Unique paths in an \(m \times n\) grid (recursive + memo)  
25. Edit Distance (recursive + memoized)  
26. Longest Common Subsequence (recursive + memoized)  

27. Palindrome Partitioning I (all partitions into palindromes)  
28. Partition to K equal‑sum subsets (backtracking on subsets)  

29. Design recursion trees and time complexity for at least 10 of the above problems (practice drawing the tree + recurrence)
